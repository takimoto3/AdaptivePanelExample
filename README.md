# AdaptivePanelExample

`AdaptivePanelExample` is a sample application demonstrating how to use the [AdaptivePanel](https://github.com/takimoto3/AdaptivePanel) UI library for SwiftUI.

## Features

This sample app showcases three primary usage patterns of `AdaptivePanel`:

- **Basic Detent**: Customizing panel heights using `CustomPanelDetent` and controlling background interactions.
- **Item-based Presentation**: Presenting panels based on `Identifiable` data with dynamic content updates.
- **Selection Control**: Programmatically switching panel heights using bound selection states.

## Requirements

- iOS 18.6+
- Xcode 16.0+
- Swift 6.0+

## Getting Started

1. Clone this repository.
2. Open `AdaptivePanelExample.xcodeproj` in Xcode.
3. Ensure that the `AdaptivePanel` library dependency is resolved correctly.
